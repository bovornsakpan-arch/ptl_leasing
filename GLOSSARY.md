# Glossary / คำศัพท์โปรเจกต์

> คำศัพท์และนิยามที่ใช้ในโปรเจกต์ PT Leasing
> Terms and definitions used in PT Leasing project

---

## SDLC & Process Terms

| Term (EN) | คำ (TH) | นิยาม / Definition |
|-----------|---------|-------------------|
| Acceptance Criteria | เกณฑ์การยอมรับ | เงื่อนไขที่ต้องผ่านก่อนถือว่า feature/story เสร็จสมบูรณ์ |
| ADR | - | Architecture Decision Record — บันทึกการตัดสินใจด้าน architecture |
| Backlog | - | รายการงานที่รอดำเนินการ |
| BRD | - | Business Requirements Document — เอกสารความต้องการทางธุรกิจ |
| Change Request (CR) | คำขอเปลี่ยนแปลง | การขอเปลี่ยนแปลง feature ที่มีอยู่แล้ว |
| Epic | - | กลุ่มของ stories ที่รวมกันเป็น feature ใหญ่ |
| Frontmatter | - | YAML metadata ที่อยู่ด้านบนของไฟล์ Markdown |
| Gap Analysis | การวิเคราะห์ช่องว่าง | การเปรียบเทียบ As-Is กับ To-Be เพื่อหาสิ่งที่ต้องพัฒนา |
| Impact Analysis | การวิเคราะห์ผลกระทบ | การประเมินผลกระทบของการเปลี่ยนแปลงต่อระบบ |
| SLA | - | Service Level Agreement — ข้อตกลงระดับบริการ/กำหนดเวลา |
| SRS | - | Software Requirements Specification — เอกสารข้อกำหนดซอฟต์แวร์ |
| Story | User Story | การอธิบาย feature จากมุมมองผู้ใช้งาน |
| Story Points | - | หน่วยวัดความซับซ้อนของ story |
| Sprint | - | รอบการพัฒนา (โดยทั่วไป 2 สัปดาห์) |
| Traceability | การติดตามเส้นทาง | ความสัมพันธ์ระหว่าง requirement → design → story → test → code |

## Leasing Business Terms / คำศัพท์ธุรกิจลีสซิ่ง

| Term (EN) | คำ (TH) | นิยาม / Definition |
|-----------|---------|-------------------|
| Leasing | ลีสซิ่ง | สัญญาเช่าทางการเงิน ผู้เช่าได้ใช้สินทรัพย์โดยไม่ได้เป็นเจ้าของ |
| Financial Lease | เช่าการเงิน | สัญญาเช่าที่ผู้เช่าแบกรับความเสี่ยงและผลตอบแทนจากสินทรัพย์ |
| Operating Lease | เช่าดำเนินงาน | สัญญาเช่าระยะสั้น ผู้ให้เช่ายังคงเป็นเจ้าของสินทรัพย์ |
| Lessee | ผู้เช่า | บุคคล/นิติบุคคลที่เช่าสินทรัพย์ |
| Lessor | ผู้ให้เช่า | บริษัทลีสซิ่งที่เป็นเจ้าของสินทรัพย์และให้เช่า |
| Asset | สินทรัพย์ | สิ่งที่นำมาให้เช่า เช่น รถยนต์ เครื่องจักร อุปกรณ์ |
| Contract | สัญญา | สัญญาเช่าระหว่างผู้ให้เช่าและผู้เช่า |
| Down Payment | เงินดาวน์ | เงินที่ผู้เช่าจ่ายล่วงหน้าก่อนเริ่มสัญญา |
| Installment | งวดผ่อนชำระ | การจ่ายเงินเป็นงวด ๆ ตามที่กำหนดในสัญญา |
| Interest Rate | อัตราดอกเบี้ย | ร้อยละของดอกเบี้ยที่คิดจากมูลค่าสัญญา |
| Residual Value | มูลค่าซาก | มูลค่าของสินทรัพย์ที่เหลือเมื่อสิ้นสุดสัญญา |
| Balloon Payment | ค่างวดก้อนสุดท้าย | การจ่ายเงินก้อนใหญ่ในงวดสุดท้ายของสัญญา |
| Credit Scoring | การประเมินเครดิต | กระบวนการประเมินความน่าเชื่อถือทางการเงินของผู้เช่า |
| KYC | รู้จักลูกค้า (Know Your Customer) | กระบวนการยืนยันตัวตนลูกค้าตามข้อกฎหมาย |
| AML | ต่อต้านการฟอกเงิน (Anti-Money Laundering) | กระบวนการป้องกันการฟอกเงิน |
| NPL | สินเชื่อที่ไม่ก่อให้เกิดรายได้ (Non-Performing Loan) | สัญญาที่ผิดนัดชำระ |
| Collateral | หลักทรัพย์ค้ำประกัน | สินทรัพย์ที่ใช้ค้ำประกันในสัญญาเช่า |
| Amortization | การตัดจำหน่าย | การลดยอดหนี้ตามแผนการชำระ |
| IRR | อัตราผลตอบแทนภายใน (Internal Rate of Return) | อัตราผลตอบแทนจากการลงทุนในสัญญาเช่า |
| DSR | อัตราส่วนภาระหนี้ต่อรายได้ (Debt Service Ratio) | สัดส่วนของภาระหนี้ต่อรายได้ของผู้เช่า |
| Tenor | ระยะเวลาสัญญา | ระยะเวลาทั้งหมดของสัญญาเช่า |
| Early Termination | การบอกเลิกสัญญาก่อนกำหนด | การยุติสัญญาก่อนครบกำหนด |
| Refinancing | การรีไฟแนนซ์ | การปรับโครงสร้างหนี้ใหม่ |

## Technical Terms / คำศัพท์เทคนิค

| Term (EN) | คำ (TH) | นิยาม / Definition |
|-----------|---------|-------------------|
| API | - | Application Programming Interface — ช่องทางสื่อสารระหว่างระบบ |
| Backoffice | ระบบหลังบ้าน | ระบบที่ใช้งานภายในองค์กร ไม่ใช่ frontend ที่ลูกค้าเห็น |
| CI/CD | - | Continuous Integration/Continuous Deployment — กระบวนการ deploy อัตโนมัติ |
| DPIA | - | Data Protection Impact Assessment — การประเมินผลกระทบต่อการคุ้มครองข้อมูล |
| RBAC | - | Role-Based Access Control — การควบคุมสิทธิ์ตาม role |
| SSO | - | Single Sign-On — การ login ครั้งเดียวใช้ได้หลายระบบ |
| MFA | - | Multi-Factor Authentication — การยืนยันตัวตนหลายขั้นตอน |
| PII | ข้อมูลส่วนบุคคล | Personally Identifiable Information |
| PDPA | พ.ร.บ.คุ้มครองข้อมูลส่วนบุคคล | Personal Data Protection Act (Thailand) |
| Audit Log | บันทึกการตรวจสอบ | บันทึกทุกการกระทำในระบบเพื่อการตรวจสอบ |
