# ยินดีต้อนรับสู่ PT Leasing Backoffice

> คู่มือการใช้งานระบบ PT Leasing Backoffice สำหรับเจ้าหน้าที่และผู้จัดการ

---

## ระบบนี้คืออะไร?

**PT Leasing Backoffice** คือระบบจัดการสัญญาเช่าสำหรับเจ้าหน้าที่และผู้จัดการของบริษัท ครอบคลุมการทำงานหลัก ดังนี้:

- **จัดการสัญญาเช่า** — สร้าง แก้ไข อนุมัติ และติดตามสัญญา
- **จัดการลูกค้า** — ลงทะเบียน KYC และดูประวัติ
- **ประมวลการชำระเงิน** — บันทึกและติดตามการชำระ
- **รายงาน** — ดูภาพรวมและ export รายงาน
- **การแจ้งเตือน** — รับแจ้งเตือนสำคัญทาง email
