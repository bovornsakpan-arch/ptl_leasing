# เริ่มต้นใช้งาน

> คู่มือการเริ่มต้นใช้งานระบบ PT Leasing Backoffice

---

## ความต้องการของระบบ

ก่อนใช้งาน ตรวจสอบให้แน่ใจว่าคุณมี:

- **Browser**: Chrome 90+, Firefox 90+, หรือ Edge 90+
- **บัญชีผู้ใช้**: ได้รับจาก System Admin
- **รหัสผ่าน**: ได้รับทาง email เมื่อสร้าง account

---

## การเข้าสู่ระบบ (Login)

### ขั้นตอนที่ 1: เปิดหน้า Login
เปิด browser และไปที่: `https://ptleasing.snocko-tech.com`

### ขั้นตอนที่ 2: กรอกข้อมูล
1. กรอก **Email** ของคุณ (เช่น name@snocko-tech.com)
2. กรอก **รหัสผ่าน**
3. คลิกปุ่ม **เข้าสู่ระบบ**

### ขั้นตอนที่ 3: เปลี่ยนรหัสผ่านครั้งแรก
เมื่อ login ครั้งแรก ระบบจะให้เปลี่ยนรหัสผ่าน:
- ความยาว: อย่างน้อย 10 ตัวอักษร
- ต้องมี: ตัวพิมพ์ใหญ่, ตัวพิมพ์เล็ก, ตัวเลข, และอักขระพิเศษ

---

## การออกจากระบบ (Logout)

คลิกที่ชื่อของคุณ (มุมบนขวา) แล้วเลือก **ออกจากระบบ**

> **หมายเหตุ**: ระบบจะ logout อัตโนมัติหลังไม่ได้ใช้งาน 30 นาที

---

## หน้าจอหลัก (Dashboard)

หลัง login สำเร็จ คุณจะเห็นหน้า Dashboard ซึ่งแสดง:
- สรุปสัญญาที่รับผิดชอบ
- การแจ้งเตือนสำคัญ
- งานที่รอดำเนินการ

---

## การนำทาง (Navigation)

เมนูด้านซ้ายมีเมนูหลักดังนี้:

| เมนู | ฟังก์ชัน |
|------|---------|
| Dashboard | หน้าหลัก สรุปภาพรวม |
| สัญญา | จัดการสัญญาเช่า |
| ลูกค้า | จัดการข้อมูลลูกค้า |
| การชำระ | บันทึกและติดตามการชำระ |
| รายงาน | รายงานต่าง ๆ (Manager เท่านั้น) |
| การตั้งค่า | ตั้งค่า profile ของคุณ |

---

## ถ้า Login ไม่ได้

| ปัญหา | วิธีแก้ |
|-------|---------|
| ลืมรหัสผ่าน | คลิก "ลืมรหัสผ่าน" ใต้ปุ่ม Login |
| Account ถูก lock | ติดต่อ Admin: admin@snocko-tech.com |
| ไม่มี account | ขอจาก Manager หรือ HR |
