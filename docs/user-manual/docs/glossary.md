# คำศัพท์ / Glossary

> คำศัพท์สำหรับผู้ใช้งานระบบ PT Leasing Backoffice

---

| คำศัพท์ | ความหมาย |
|--------|---------|
| สัญญาเช่า | ข้อตกลงระหว่างบริษัทและลูกค้าสำหรับการเช่าสินทรัพย์ |
| ผู้เช่า (Lessee) | ลูกค้าที่ทำสัญญาเช่า |
| ค่างวด | เงินที่ลูกค้าชำระเป็นรายเดือนตามสัญญา |
| สถานะสัญญา Active | สัญญาที่กำลังใช้งานอยู่ |
| สถานะสัญญา Expired | สัญญาที่หมดอายุแล้ว |
| สถานะสัญญา Terminated | สัญญาที่ถูกยกเลิกก่อนกำหนด |
| KYC | Know Your Customer — กระบวนการตรวจสอบตัวตนลูกค้า |
| Credit Score | คะแนนเครดิตของลูกค้า ใช้ประเมินความเสี่ยง |
| NPL | สัญญาที่ผิดนัดชำระ (Non-Performing Loan) |
| Dashboard | หน้าแสดงภาพรวมของงานทั้งหมด |
| Manager | ผู้จัดการที่มีสิทธิ์อนุมัติสัญญา |
| Officer | เจ้าหน้าที่ที่สร้างและจัดการสัญญา |
