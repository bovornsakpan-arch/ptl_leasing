# FAQ / คำถามที่พบบ่อย

---

## การเข้าสู่ระบบ

**Q: ลืมรหัสผ่านทำอย่างไร?**
A: คลิก "ลืมรหัสผ่าน" ที่หน้า Login แล้วกรอก email ระบบจะส่ง link รีเซ็ตรหัสผ่านให้

**Q: Account ถูก lock ทำอย่างไร?**
A: ติดต่อ Admin: admin@snocko-tech.com หรือโทร [เบอร์ support]

**Q: ระบบ logout เองทำไม?**
A: ระบบจะ logout อัตโนมัติหลังไม่ได้ใช้งาน 30 นาที เพื่อความปลอดภัย

---

## การใช้งานทั่วไป

**Q: ฉันไม่เห็นเมนู [X] ทำอย่างไร?**
A: เมนูบางอย่างแสดงตาม role เช่น รายงานแสดงเฉพาะ Manager หากต้องการสิทธิ์เพิ่ม ติดต่อ Manager ของคุณ

**Q: ข้อมูลไม่อัปเดต ต้องทำอย่างไร?**
A: ลอง refresh หน้า (F5) หรือ logout แล้ว login ใหม่

---

## ปัญหาเทคนิค

**Q: ระบบช้ามาก ทำอย่างไร?**
A: ลองปิดแล้วเปิด browser ใหม่ ถ้ายังช้า แจ้ง support

**Q: เกิด error "เซสชันหมดอายุ" ทำอย่างไร?**
A: Login ใหม่อีกครั้ง
